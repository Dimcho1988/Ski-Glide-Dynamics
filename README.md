# onFlows — Модулиране на скоростта при ски / ролки

## Какво прави това приложение

- Чете TCX файлове от ски бягане / ролки
- Оценява ефективен коефициент на триене µ_eff от спусканията
- Модулира скоростта към избрана референтна плъзгаемост µ_ref
- Показва сравнение между реална и модулирана скорост

## Стартиране локално

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
