# onFlows — Демонстрация на модел за коефициент на триене (v2)

Това е минимално Streamlit приложение, което имплементира модела за
коефициент на триене и модулирана скорост с фиксирана референтна активност.

В тази v2 версия:

- Добавено е автоматично **премахване на дублирани времеви точки (time_s)**,
  което решава грешката `cannot reindex on an axis with duplicate labels`
  при някои TCX файлове (особено при Garmin устройства).

## Файлове

- `streamlit_app.py` — основно приложение (UI).
- `friction_model.py` — функции за обработка на данните и модела (v2, с фиксация).
- `requirements.txt` — зависимости за инсталиране.

## Стартиране (локално)

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
